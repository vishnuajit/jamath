 Theme Options
--------------------------------------------------------------------------
We provide two ways to customize your site. On your WordPress Dashboard

Appearance > Theme Options and Appearance > Customize

To produce the layout on screenshot:
1. Create a page called Home and use page template "Home Template"
2. Customize the Home Page using Theme Options
3. Set the home as static front page in Settings > Reading
